# Ron’s Retirement Finance Model V1

A comprehensive, browser-based retirement planning and tax modeling tool.

Models retirement year-by-year with detailed calculations for income, spending, Roth conversions, Social Security, taxes, and IRMAA. Built for realistic scenario testing and long-term planning.
